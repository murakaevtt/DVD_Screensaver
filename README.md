# A regular fullscreen screensaver made with pygame. There's nothing more to say.

To run a program, only download 'screensaver.exe' from releases, and run it)

Launch from source ----------------------

**Windows:**
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
python main.py

**Linux/Mac:**
python -m venv .venv
.venv\bin\activate
pip install -r requirements.txt
python main.py
